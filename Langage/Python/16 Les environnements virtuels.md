# Les environnements virtuels

Créer un environnement virtuel:
> python -m venv venv

Activer un environnement virtuel:

LINUX
> source ./venv/bin/activate

WINDOWS
Activer un environnement virtuel:
> source ./venv/Scripts/activate.bat

Désactiver un environnement virtuel:
> deactivate
